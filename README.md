# Provax

<p>Created during the pandemic, Provax is a dummy mobile application (and now a web application) that imitates Co-Win , a vaccination portal for Covid-19 related vaccines.Provax makes use of Cloud Firestore, a NoSQL cloud database by Firebase and Google.User authentication and hosting is facilitated by Firebase.
The vaccination centre data is provided by https://apisetu.gov.in/</p>

<p>Users are provided with the option to choose either covaxin or covishield and book a slot at a vaccination centre providing that vaccine.The vaccination centres are located by entering their pincode. After successfully getting both the doses of a vaccine, the users are provided with a custom vaccination certificate.</p>
